### Project Setuo Guide 

1. Clone the project 

```
git clone https://github.com/aashikofsach/ShopCart-E-Commerce-Backend.git

```

2. Move into the directory 

```
cd ShopCart-E-Commerce-Backend

```

3. Install Node-Modules 

```
npm i 

```
4. setup the .env file at the root directory and add the PORT as :

```
PORT : YOUR_PORT_NUMBER

```



5. Run the Project 

```
npm start 

```

6. service deployed on Render and DB service using from freesqldatabase.com

```
https://dashboard.render.com/

https://www.freesqldatabase.com/account/

```
